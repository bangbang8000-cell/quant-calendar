## 1. 产品定位

量化选股日历是一个面向 A 股量化投资者的决策辅助工具，基于美林时钟宏观经济周期理论，整合多策略选股信号、AI 评估、回测分析和数据可视化。

### 1.1 核心价值主张

| 能力层 | 描述 |
|--------|------|
| **感知层** | 美林时钟五维度宏观评分，实时识别经济周期阶段 |
| **信号层** | 多因子策略（动量/反转/成长/价值）选股，共识榜 + 股票池 |
| **决策层** | AI 多模型串行评估 + 信号归因 + 狙击点 + 仓位建议 |
| **验证层** | 回测引擎 + 评估历史对比 + 操作检查清单 |
| **协作层** | 飞书定时推送 + 多用户权限管理 |

### 1.2 用户

**当前用户（v2.0.0）**：独立量化交易者，单用户使用，主要场景为每日选股信号查看和 AI 辅助评估。

**目标用户（v2.4.0 后）**：

| 角色 | 场景 | 需求 |
|------|------|------|
| 独立交易者 | 决策仪表盘、AI 评股、仓位管理 | 当前已覆盖 |
| 量化研究员 | 策略开发、回测验证、多策略对比 | v2.3.0 回测增强 |
| 团队管理者 | 多用户权限、策略共识、推送通知 | v2.4.0 协作特性 |
| 访客/客户 | 只读查看策略表现和市场状态 | 已支持访客模式 |
