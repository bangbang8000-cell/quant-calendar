## 五、执行建议

### 优先级 P0（用户高频可见）

1. JS ElMessage 去 emoji 前缀（15 处）— 每次操作都会看到
2. JS ElMessage 去感叹号（5 处）
3. 前端 UI 文案去营销腔（~10 处，详见 UI-COPY-AUDIT.md）

### 优先级 P1（用户偶尔可见）

4. 前端 UI 说明书式文字（~8 处）
5. Python API 返回消息（7 处）

### 优先级 P2（仅开发者可见）

6. Python 注释/docstring（8 处）
7. 前端 emoji 装饰清理（~8 处）

### 不做修改

- AI prompt 模板（给 LLM 看的，不影响用户）
- 技术注释（`# v2.1: 时间驱动` 等，有价值）
- 日志消息（`logger.info(...)` 等，不出现在 UI）
